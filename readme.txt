== Installation ==
1. Upload `rhx-woo-carousel` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>[rhxwoo-carousel]</strong> in the post/page" to display product carousel.
